# Multi-Stopwatch Timer

A web-based stopwatch application that allows you to manage up to 20 stopwatches simultaneously with keyboard shortcuts.

## Features

- **Multiple Stopwatches**: Create and manage up to 20 independent stopwatches
- **Custom Names**: Give each stopwatch a meaningful name
- **Keyboard Shortcuts**: Assign keyboard shortcuts to quickly start/stop specific stopwatches
- **Batch Controls**: Start all, stop all, reset all, or clear all stopwatches at once
- **CSV Export**: Export all stopwatch data to a CSV file for analysis
- **Auto-Save**: All data is automatically saved to your browser's local storage
- **Responsive Design**: Works on desktop, tablet, and mobile devices

## Usage

1. Click "Add Stopwatch" to create a new stopwatch
2. Click on the stopwatch name to edit it
3. Use Start/Stop buttons to control individual stopwatches
4. Click "Set Key" to assign a keyboard shortcut
5. Use the top control buttons for batch operations
6. Click "Export to CSV" to download your stopwatch data

## Development

This project uses Angular 19.

```bash
npm install
npm start
```
